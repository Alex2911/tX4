Chatzilla

¿Qué es ChatZilla?

ChatZilla es un cliente de IRC (Internet Relay Chat) para Firefox sencillo, extenso y fácil de usar (se puede usar en Mac, Windows y Linux).

Instalación

Visita la página web de ChatZilla con Firefox. Instálalo y reinicia Firefox.

Conectarse y entrar a un canal

Vete a Herramientas en el menú y selecciona "ChatZilla" para iniciarlo.

Escribe /server irc.travian.com (no importa si es .com, .net o .de) en la línea de mensajes para conectarte a la red de IRC de Travian.

Cuando te hayas conectado escribe /nick el_nick_que_quieras_ponerte para cambiar tu nick.

Luego debes escribir /join #travian para entrar al canal internacional de Travian o /join #travian.net para entrar al canal oficial de Travian.net.

Cuando veas una lista de varios nombre de usuario a la izquierda, habrás entrado al canal. Felicidades.
